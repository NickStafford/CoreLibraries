Layout Fields

Layout Class  See Also

The following tables list the members exposed by the Layout type.

Fields
Name Description
Alignment The alignment.
Default The default layout (as specified by the current layout writer).
Empty The empty layout makes no changes.
FirstLineIndentSize The first line indent size.
HyphenChar The hyphenation character is used to split words.
Hyphenate Whether to add a hyphen when splitting words.
IndentChar The indent character (is repeated IndentSize times).
IndentSize The indent size.
RightMarginSize The size of any right margin.
SplitLength If this value is zero then words will only be split if they are longer than an entire line; otherwise the word will only be split at a point that has at least this number of character's on either side of the split (recommended value is 3, to handle splitting of words with apostrophes).
TabChar The tab character is used to fill to next tab stop.
TabSize The tab size, used to produce tabs when the layout doesn't support tab stops.
TabStops The tab stops, only valid for Left and None.
Width The layout width.
WrapMode The line wrap mode

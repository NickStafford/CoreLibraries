Parameters

flags

Type: TEnum

The flags to split.

includeImplicit

Type: bool

If set to true includes implicit values of flag enums.

By default this is set to false.

includeCombinations

Type: bool

If set to true will split into the minimum number of flags (allows combinations).

By default this is set to false.

Return Value

The result of the split flags.

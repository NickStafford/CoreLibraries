SecureIdentifier Members

SecureIdentifier Class  Constructors  Methods  Fields  Properties  See Also

A secure identifier that can be used safely in URLs, file names, etc. (as it's base32 encoded) and can be more secure than a GUID as it uses a cryptographically secure random number generator. It also has a built in hash to ensure only certain identifiers are valid, making it capable of rejecting invalid identifiers (for example during a DDOS attack).

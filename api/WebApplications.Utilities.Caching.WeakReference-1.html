    public WeakReference(T target, bool trackResurrection = false)
    Parameters
    Type Name Description
    T target

    The target, which is the underlying object being referenced.
    Boolean trackResurrection

    If set to true then the weak reference will be long.
    Remarks

    If trackResurrection is true then a long weak reference is created otherwise a short weak reference is created. A long reference is retained after the objects Finalize method is called up until it is collected.

    If the object type doesn't offer a Finalize method then short and long references behave the same.

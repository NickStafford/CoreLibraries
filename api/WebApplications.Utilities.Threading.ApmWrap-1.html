    Struct ApmWrap<T>

    A light-weight struct with the ability to associate an arbitrary piece of data of type T with any IAsyncResult object. When the asynchronous operation completes this data can be retrieved to complete processing.

    T

    The type of the data to embed.
    Remarks

    This is typically used when you are implementing code that wraps an asynchronous operation and you wish to add some context or state of your own to complete the wrapping.

    Based on Jeff Ritcher's wintellect threading libraries.

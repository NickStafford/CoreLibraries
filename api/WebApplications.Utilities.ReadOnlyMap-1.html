    The item type.
    Remarks

    The ReadOnlyMap<T> extends a List<T> of IReadOnlyList<T>, and so can be initialized and manipulated as a list. When in use it can be case to a IReadOnlyList<T> of T or an ICollection, at which point it acts like a single collection mapping requests for calls onto the underlying data.

    The ReadOnlyOffsetMap<T> can be used where you wish to map onto a single data source.

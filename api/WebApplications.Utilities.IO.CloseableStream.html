    CloseableStream(Stream, Action, Func<Exception, Boolean>)

    Create a stream that will invoke the specified onClose action when the stream is closed/disposed.

    Declaration
    [PublicAPI]
public CloseableStream(Stream stream, Action onClose, Func<Exception, bool> onError = null)
    Parameters
    Type Name Description
    Stream stream

    The stream to wrap
    Action onClose

    The action to call after the specified stream is closed.
    Func<Exception, Boolean> onError

    The optional function to call if the underlying stream errors on close (onClose will still be called after onError), return true to suppress the exception.
    Exceptions
    Type Condition
    ArgumentNullException

    stream is null.
    ArgumentNullException

    onClose is null.

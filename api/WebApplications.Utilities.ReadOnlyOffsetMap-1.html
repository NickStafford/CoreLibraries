    Remarks

    The ReadOnlyOffsetMap<T> extends a List<T> of Mapping, and so can be initialized and manipulated as a list. When in use it can be cast to a IReadOnlyList<T> of T or an ICollection, at which point it acts like a single collection mapping requests for calls onto the underlying data.

    The ReadOnlyMap<T> can be used where you wish to map onto multiple data sources.

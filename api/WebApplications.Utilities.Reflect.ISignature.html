    Close(Type[], Type[])

    Closes the signature with the specified concrete generic types.

    Declaration
    ISignature Close(Type[] typeClosures, Type[] signatureClosures)
    Parameters
    Type Name Description
    Type[] typeClosures

    The types required to close the current type.
    Type[] signatureClosures

    The types required to close the signature.
    Returns
    Type Description
    ISignature

    A closed signature, if possible; otherwise null.
    Remarks

    If signature closure is unsupported this method should return null.

    The closure arrays are ordered and contain the same number of elements as their corresponding generic arguments. Where elements are null a closure is not required.

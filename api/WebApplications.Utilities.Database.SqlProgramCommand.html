    AdditionalCancellationTime

    The additional time given to a command before a cancellation is triggered.

    Declaration
    public static readonly TimeSpan AdditionalCancellationTime
    Field Value
    Type Description
    TimeSpan
    Remarks

    All commands, even manually disposed ones, will be cleaned up if the CommandTimeout is greater than Zero and the CommandTimeout plus the AdditionalCancellationTime has elapsed.

    This ensures that resources don't leak over time, by badly written consumers.

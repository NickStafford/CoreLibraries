    IsInvariant(CultureInfo)

    Determines whether or not the CultureInfo is the invariant culture. (http://msdn.microsoft.com/en-us/library/4c5zdc6a.aspx)

    Declaration
    public static bool IsInvariant(this CultureInfo cultureInfo)
    Parameters
    Type Name Description
    CultureInfo cultureInfo

    The culture info.
    Returns
    Type Description
    Boolean

    Returns true if the specified cultureInfo is invariant; otherwise returns false.
    Remarks

    The invariant culture is culture-insensitive, it is useful for when culture-specific presentation isn't required/needed.

    ObjectRecord(RecordSetDefinition, Boolean, Double, Func<Int32, Object>[], Int32)

    Initializes a new instance of the ObjectRecord class.

    Declaration
    public ObjectRecord(RecordSetDefinition recordSetDefinition, bool randomData = false, double nullProbability = 0.1, Func<int, object>[] columnGenerators = null, int rowNumber = 1)
    Parameters
    Type Name Description
    RecordSetDefinition recordSetDefinition

    The table definition.
    Boolean randomData

    if set to true fills columns with random data; otherwise fills them with their default values.
    Double nullProbability

    The probability of a column's value being set to SQL null (0.0 for no nulls) - this is only applicable is randomData is set to true [Defaults to 0.1 = 10%].
    Func<Int32, Object>[] columnGenerators

    The column generators is an array of functions that generate a value for each column, if the function is null for a particular index then a random value is generated, if it is not null then the function is used. The function takes the current row number as it's only parameter and must return an object of the correct type for the column.
    Int32 rowNumber

    The optional row number to pass to the generator.
    Exceptions
    Type Condition
    ArgumentException

    Thrown if the number of column generators exceeds the number of columns in the record set definition.

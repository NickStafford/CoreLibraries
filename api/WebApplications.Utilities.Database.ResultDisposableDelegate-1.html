    Delegate ResultDisposableDelegate<T>

    A delegate that accepts any method that accepts a SqlDataReader and returns a result.

    Namespace:WebApplications.Utilities.Database
    Assembly:WebApplications.Utilities.Database.dll
    Syntax
    public delegate T ResultDisposableDelegate<out T>(SqlDataReader reader, IDisposable disposable);
    Parameters
    Type Name Description
    SqlDataReader reader

    The reader.
    IDisposable disposable

    The disposer that can be used to clean up manually.
    Returns
    Type Description
    T

    The result.
    Type Parameters
    Name Description
    T

    The result type.
    Remarks

    The disposable can be used to dispose the reader, it's associated SqlCommand and SqlConnection, by calling the Dispose() method when ready.

    This is most useful when retrieving a stream from the reader and can be used with a CloseableStream.

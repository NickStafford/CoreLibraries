    IEnumerable<KeyValuePair<String, String>> dictionary

    The key value pairs.
    Remarks

    This is used for deserializing Log entries - it does not result in logs being added!

    To add logs use Add(String, Object[]) instead.

    You can create partial logs, however the context must contain at least the GuidKey, and be a valid CombGuid.

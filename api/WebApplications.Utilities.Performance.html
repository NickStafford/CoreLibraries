    Namespace WebApplications.Utilities.Performance

    The WebApplications.Utilities.Performance NuGet makes it easy to access to instance based performance counters and timers, includes the ability to set up performance counters automatically as part of deployments (using the included PerfSetup tool), or allows performance data to be collected in the absence of performance counters.

    Classes

    Counter

    A counter which also tracks the current rate, by keeping track of the last MaximumSamples instances the counter was incremented.

    CriticalPerfTimer

    Times an operation using a performance timer, tracks incidents of timer exceeding a warning and/or critical duration.

    PerfCategory

    Performance category helper.

    PerfCounter

    Performance counters used to count operations.

    PerfCounterInfo

    Provides read-only access to a PerfCategory information.

    PerfCounterInfo<T>

    Provides read-only access to a PerfCategory information.

    PerformanceCounterExtensions

    Extensions for performance counters.

    PerfTimer

    Times an operation using a performance timer.

    RegionTimer

    Used to allow the timing of a region of code for a PerfTimer.

    Timers

    A counter of RegionTimer which also tracks the current rate, by keeping track of the last MaximumSamples instances the counter was incremented.

    Delegates

    RegionTimerDisposedDelegate

    Called when a RegionTimer is Dispose() (i.e. stopped).

    Namespace WebApplications.Utilities.Database

    The WebApplications.Utilities.Database NuGet is used for connecting enterprise level .Net applications to SQL Server databases, and allows for easy interaction with stored procedures, load balancing and concurrency control. It also ensures that deployed schemas are valid based on the required usage.

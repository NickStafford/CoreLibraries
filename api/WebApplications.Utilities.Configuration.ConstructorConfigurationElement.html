    GetConstructor<T>()

    Gets the constructor as an action, which can be used to create instances of the specified object.

    Declaration
    public Func<T> GetConstructor<T>()
    Returns
    Type Description
    Func<T>

    Returns a Func<TResult> that can be used to create an instance of the specified object.
    Type Parameters
    Name Description
    T

    The type to get the constructor for.
    Remarks

    This should only be called once as it uses reflection. Store the resulting Func<TResult> where possible and use it to repeatedly create new instances.

    Exceptions
    Type Condition
    InvalidOperationException

    The configuration system found multiple potential constructor matches.

    -or-

    The configuration system couldn't find a constructor with the relevant parameters.

    -or-

    The configuration system cannot assign the created type to the return type.
    ConfigurationErrorsException

    The required type property is not set.

    GetInstance<T>()

    Gets an instance of the Object.

    Declaration
    public T GetInstance<T>()
    Returns
    Type Description
    T

    An instance of an object of type T.
    Type Parameters
    Name Description
    T
    Remarks

    This should only be called once as it calls the GetConstructor<T>() method each time. Instead you should use GetConstructor<T>() where possible and store the resultant Func<TResult>, which can then be called repeatedly to create new instances.

    Exceptions
    Type Condition
    InvalidOperationException

    The configuration system found multiple potential constructor matches.

    -or-

    The configuration system couldn't find a constructor with the relevant parameters.

    -or-

    The configuration system cannot assign the created type to the return type.
    ConfigurationErrorsException

    The required type property is not set.

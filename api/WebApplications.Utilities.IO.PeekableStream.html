    Class PeekableStream

    PeekableStream wraps a Stream and can be used to peek ahead in the underlying stream, without consuming the bytes. In other words, doing Peek() will allow you to look ahead in the stream, but it won't affect the result of subsequent Read() calls.

    This is sometimes necessary, e.g. for peeking at the magic number of a stream of bytes and decide which stream processor to hand over the stream.

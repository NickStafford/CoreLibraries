    Namespace WebApplications.Utilities.Cryptography

    The WebApplications.Utilities.Cryptography NuGet provides utilities for easier use and configuration of cryptographic services in enterprise solutions.

    Classes

    AsymmetricCryptographyProvider

    Base class for all asymmetric cryptographic providers.

    Base32EncoderDecoder

    Extension methods supporting clean Base32 encoding, with custom digits (skips easily confused characters).

    CryptographyExtensions

    Extensions for hashing algorithms.

    CryptographyProvider

    Base class for all cryptography providers.

    CryptographyProvider.CryptoTransform<T>

    Class CryptographyProvider.CryptoTransform<T> is a base implementation of the ICryptoTransform interface.

    EncryptedString

    Wraps together encrypted and unencrypted versions of a string, only encrypting/decrypting when necessary.

    HashingCryptographyProvider

    Base class for all hashing cryptographic providers.

    KeyedHashingCryptographyProvider

    Base class for all hashing cryptographic providers that have a key.

    PBKDF2<T>

    Implements password-based key derivation functionality using PBKDF2.

    RandomCryptographyProvider

    Base class for all random number generating cryptographic providers.

    RSACryptographyProvider

    Implements the RSA algorithm as an AsymmetricCryptographyProvider.

    SecureIdentifier

    A secure identifier that can be used safely in URLs, file names, etc. (as it's base32 encoded) and can be more secure than a GUID as it uses a cryptographically secure random number generator. It also has a built in hash to ensure only certain identifiers are valid, making it capable of rejecting invalid identifiers (for example during a DDOS attack).

    SymmetricCryptographyProvider

    Base class for all symmetric cryptographic providers.

    XmlCryptoExtensions

    Extensions for Xml cryptography.

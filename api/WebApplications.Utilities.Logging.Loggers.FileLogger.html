    public FileLogger(string name, string directory = null, long maxLog = 1000L, TimeSpan maxDuration = null, LoggingLevels validLevels = LoggingLevels.All | LoggingLevels.AtLeastCritical | LoggingLevels.AtLeastError | LoggingLevels.AtLeastWarning | LoggingLevels.AtLeastSystemNotification | LoggingLevels.AtLeastNotification | LoggingLevels.AtLeastInformation | LoggingLevels.Emergency | LoggingLevels.Critical | LoggingLevels.Error | LoggingLevels.Warning | LoggingLevels.SystemNotification | LoggingLevels.Notification | LoggingLevels.Information | LoggingLevels.Debugging, FormatBuilder format = null, FormatBuilder fileNameFormat = null, string extension = null, uint buffer = 65536U, bool autoFlush = false)
    Parameters
    Type Name Description
    String name

    The filename.
    String directory

    The directory to log to (default to entry assembly directory).
    Int64 maxLog

    The maximum number of log items in a single log file.

    By default this is set to 1,000.
    TimeSpan maxDuration

    The maximum time period that a single log file can cover.
    LoggingLevels validLevels

    The valid log levels.

    By default allows LoggingLevels.
    FormatBuilder format

    The log format (default to "Verbose,Xml").
    FormatBuilder fileNameFormat

    The filename format - where {DateTime} is the creation date time.

    By default the format is "{ApplicationName}-{DateTime:yyMMddHHmmssffff}".
    String extension

    The file extension.

    By default this is set to use "log".
    UInt32 buffer

    The buffer.
    Boolean autoFlush

    if set to true [auto flush].
    Exceptions
    Type Condition
    LoggingException

    maxLog was less than 10, which would result in too many log files to be created.

    -or-

    maxDuration was less than 10 seconds, which would result in too many log files to be created.

    -or-

    directory was either IsNullOrWhiteSpace(String).

    -or-

    The fileNameFormat string was either IsNullOrWhiteSpace(String).

    -or-

    An error occurred trying to access the directory.

    -or-

    extension was more than 5 characters long.

    -or-

    The fileNameFormat led to an invalid path or created a path that references the wrong directory.

    -or-

    File path contained GetInvalidPathChars().

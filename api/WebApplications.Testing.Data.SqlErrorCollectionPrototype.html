    public sealed class SqlErrorCollectionPrototype : ICollection, IEnumerable
    Remarks

    To add SqlError you will probably want to create them with the SqlErrorPrototype class, which supports implicit conversion to SqlError.

    In the same way, this class supports implicit conversion to SqlErrorCollection.

    Remarks

    CurrencyInfo contains:

    • The currency ISO Code. (http://www.xe.com/iso4217.php)
    • The currency ISO Number. (http://www.xe.com/iso4217.php)
    • The exponent, which is the number of decimals available in the currency.
    • The currency's full name.
    • Associated RegionInfo.
    • Associated CultureInfo.

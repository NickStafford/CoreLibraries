    Enum ValidationFailMode

    Describes what should happen if a column/property set fails the validation check.

    Namespace:WebApplications.Utilities.Database
    Assembly:WebApplications.Utilities.Database.dll
    Syntax
    [PublicAPI]
public enum ValidationFailMode

    Fields

    Name Description
    Exception

    Throws an ArgumentOutOfRangeException if the value of the column/property fails the validation check.
    Ignore

    If the value of the column/property fails the validation check, the previous value is left in place.
    TreatAsDefault

    If the value of the column/property fails the validation check, it is replaced with the value in ValueWhenColumnIsNull (by default this is null or the default value for the type).

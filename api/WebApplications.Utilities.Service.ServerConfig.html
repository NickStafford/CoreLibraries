    Declaration
    public ServerConfig(WellKnownSidType sidType, SecurityIdentifier domainSid = null, string name = null, int maximumConnections = 1, TimeSpan heartbeat = null)
    Parameters
    Type Name Description
    WellKnownSidType sidType

    One of the enumeration of well known sid types, the value must not be LogonIdsSid. This defines who can connect to the pipe.
    SecurityIdentifier domainSid

    The domain SID. This value is required for the following WellKnownSidType values. This parameter is ignored for any other WellKnownSidType values.

    • AccountAdministratorSid
    • AccountGuestSid
    • AccountKrbtgtSid
    • AccountDomainAdminsSid
    • AccountDomainUsersSid
    • AccountDomainGuestsSid
    • AccountComputersSid
    • AccountControllersSid
    • AccountCertAdminsSid
    • AccountSchemaAdminsSid
    • AccountEnterpriseAdminsSid
    • AccountPolicyAdminsSid
    • AccountRasAndIasServersSid
    String name

    The pipe name.
    Int32 maximumConnections

    The maximum number of connections.
    TimeSpan heartbeat

    The heartbeat timespan, ensures a connection is always available, defaults to once every 5 seconds, specify a negative value to disable.

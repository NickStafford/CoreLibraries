    Namespace WebApplications.Utilities.Serialization

    The WebApplications.Utilities.Serialization NuGet provides serialization tools for .NET.

    Classes

    DelegateSerializer

    Allows serialization of delegates.

    ExtendedSerializationBinder

    Extends serialization binder to improve performance and to also allow custom type mappings to be added easily using MapType(String, String, Type). Also simplifies type and assembly names making binding across multiple versions far more reliable. (Includes fix for simplification of generic type names).

    Serialize

    A class used to assist in implementing serialization code, adds transparent support for Xml serialization (both XmlElement and XElement). Also adds easy support for mapping old types to new types, using MapType(String, String, Type).

    XElementSurrogate

    Serialization surrogate used to serialize/deserialize XElements.

    XmlElementSurrogate

    Serialization surrogate used to serialize/deserialize XmlElements.

    XmlFormatter

    The XmlFormatter class implements a custom XML Formatter, which uses the ISerializable interface. The class implements the IFormatter interface to serialize and deserialize the object to an XML representation.

    Structs

    XmlFormatter.CustomSerializationEntry

    The XmlFormatter.CustomSerializationEntry mimics the SerializationEntry class to make it possible to create our own entries. The class acts as a placeholder for a type, it's name and it's value. This class is used in the XmlFormatter class to serialize objects.

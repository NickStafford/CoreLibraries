    Implements IWriteable to produce an object that, when written, ouputs it's specified format. For example, ToString("My format") => "My format".

    FormatTextWriter

    SerialTextWriter

    Wraps a text writer so that is is synchronized by a SynchronizationContext.

    TraceTextWriter

    Implements a TextWriter for tracing. You cannot get an instance directly, but use Default to access a TraceTextWriter that is wrapped by a FormatTextWriter to provide synchronization and formatting support.

    Writeable

    Base class for implementing IWriteable and IFormattable.

    Indicates the object supports writing to a TextWriter using an optional FormatBuilder to specify the format.

    Indicates the TextWriter is synchronized with a SerializingSynchronizationContext, making it easier to group multiple writes in a block that won't be interrupted from another thread.

    Indicates the object supports writing to a TextWriter.

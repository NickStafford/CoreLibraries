    Check(TriState)

    Checks that the Object is an enum as well as whether the object is/isn't a flag (depending on the parameter specified). An exception is thrown if the object is not an enum or if the object is/isn't a flag.

    Declaration
    public static void Check(TriState isFlag = null)
    Parameters
    Type Name Description
    TriState isFlag
    • If set to Yes check that the object is an enum and that it is a flag.
    • If set to No check that the object is an enum and that it is not a flag.
    • If no value is specified then only check that the object is an enum.
    Examples
      [Flags]
  private enum Days
  {
      Mon,
      Tues,
      Wed,
      Thurs,
      Fri
  };

  ExtendedEnum<Days>.Check();               // Checks if Days is an enum. If it isn't an exception will be thrown.
  ExtendedEnum<Days>.Check(TriState.Yes);   // Also checks if Days is a flag. If it is NOT a flag or is NOT an enum then an exception will be thrown.
  ExtendedEnum<Days>.Check(TriState.No);    // Also checks if Days isn't a flag. If it IS a flag or is NOT an enum then an exception will be thrown.

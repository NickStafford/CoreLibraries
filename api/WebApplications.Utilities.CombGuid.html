    Struct CombGuid

    A Guid that encodes its creation date and can be used with SQL Server to create efficient inserts and indexes.

    CombGuid(String)

    Initializes a new instance of the CombGuid using the String specified.

    Declaration
    public CombGuid(string g)
    Parameters
    Type Name Description
    String g

    The string containing a Guid. The format can be one of the following (d represents a hex digit):

    32 continuous digits: dddddddddddddddddddddddddddddddd.

    -or-

    Groups of 8, 4, 4, 4 and 12 digits: dddddddd-dddd-dddd-dddd-dddddddddddd

    (The entire GUID can be enclosed in either braces or parenthesis.)

    -or-

    Groups of 8, 4, and 4 digits, then a subset of 8 groups of 2 digits. Each group is prefixed by "0x" or "0X" and is separated by commas. The entire GUID, as well as the subset, is enclosed in matching braces: {0xdddddddd, 0xdddd, 0xdddd,{0xdd,0xdd,0xdd,0xdd,0xdd,0xdd,0xdd,0xdd}}

    Embedded spaces and leading zeros are ignored as well as leading zeros in a group.
    Exceptions
    Type Condition
    ArgumentNullException

    g is a null.
    FormatException

    The format of g is invalid.
    OverflowException

    The format of g is invalid.

    A CombGuid containing the parsed input, which is in the specified format.
    Remarks

    The format can be:

    • "N" - 32 digits.
    • "D" - 32 digits separated by hyphens.
    • "B" - 32 digits separated by hyphens (enclosed in braces).
    • "P" - 32 digits separated by hyphens (enclosed in parenthesis).
    • "X" - Four hex values (enclosed in braces) where the fourth value is a subset of eight hex values (also enclosed in braces). ({0x00000000,0x0000,0x0000,{0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00}}

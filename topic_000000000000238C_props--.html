ISchema Properties

ISchema Interface  See Also

The following tables list the members exposed by the ISchema type.

Properties
Name Description
Guid Unique identity of the schema.
ProgramDefinitions Gets the program definitions.
ProgramDefinitionsByName Holds all the program definitions (SqlProgramDefinition) for the schema.
Schemas Gets the SQL schemas that were loaded from the database.
SchemasByID Holds all the SQL schemas (SqlSchema, using the ID as the key.
Tables Gets the table and view definitions.
TablesByName Holds all the table and view definitions (SqlTableDefinition) for the schema.
Types Gets the SQL types from the schema.
TypesByName Holds all the types for the schema, which are stored with the full name as the key and the SqlType as the value.

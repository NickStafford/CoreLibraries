LayoutWrapMode Enumeration

Describes the Layout line wrapping modes.

Namespace: WebApplications.Utilities.Formatting
Assembly: WebApplications.Utilities (in WebApplications.Utilities.dll)

Members
  Member name Description
NewLine Always adds a new line when wrapping lines, unless as the width of the writer.
NewLineOnShort Only adds a new line if the line length is less than Width, otherwise it assumes that the writer/display wraps if you reach the width.
PadToWrap Adds indent characters to the end of each line till Width is reached and lets the display handle wrapping.
PadToNewLine Adds indent characters to the end of each line till Width is reached and then adds a new line, if necessary.

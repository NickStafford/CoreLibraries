nullProbability

The probability of a column's value being set to SQL null (0.0 for no nulls) [Defaults to 0.1 = 10%].

columnGenerators

Type: Func<int, object>[]

The column generators is an array of functions that generate a value for each column, if the function is null reference (Nothing in Visual Basic) for a particular index then a random value is generated, if it is not null then the function is used. The function takes the current row number as it's only parameter and must return an object of the correct type for the column.

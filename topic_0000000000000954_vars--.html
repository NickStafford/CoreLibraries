Fields
Name Description
Alignment The alignment, if any; otherwise 0. (e.g. -3 for '{0,-3:G}'
ChildrenInternal The child chunks.
Format The format, if this is a formatting chunk and a format was specified, if any; otherwise null reference (Nothing in Visual Basic). (e.g. 'G' for '{0,-3:G}')
IsControl Control chunks are never written out when you call ToString, but can be used by consumers of a FormatBuilder to extend functionality.
IsResolved Gets a value indicating whether this instance is resolved.
Resolver Any resolver that should be used to resolve this chunk (and any nested chunks).
Tag The tag, if this is a formatting chunk, if any; otherwise null reference (Nothing in Visual Basic). (e.g. '0' for '{0,-3:G}')
Value The chunk Value, if any (will always be null reference (Nothing in Visual Basic) if IsResolved is true).

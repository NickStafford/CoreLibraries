ConsoleClient.Run Method

ConsoleClient Class  See Also

Runs the client, optionally connecting to the given service. If no service is given, or the service is invalid, the user will be prompted to select a service to connect to.

Overload List
Name Description
Run(string, NamedPipeServerInfo) Runs the client, optionally connecting to the given service. If no service is given, or the service is invalid, the user will be prompted to select a service to connect to.
Run(string, string) Runs the client asynchronously, optionally connecting to the service with the given pipe. If no pipe is given, or the pipe is invalid, the user will be prompted to select a service to connect to.
